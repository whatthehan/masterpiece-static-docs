#07 В· Rust иҝҒз§»й—ЁзҰҒпјҲL1 еҗҺпјү

иҝҷд»Ҫй—ЁзҰҒжҳҜдёҖд»ҪиҝҒз§»еҶізӯ–и®°еҪ•пјҢиҖҢдёҚжҳҜ Rust еӯҰд№ иҝӣеәҰиЎЁгҖӮе®ғиҰҒжұӮе…ҲиҜҒжҳҺй—®йўҳзЎ®е®һдҪҚдәҺеҖҷйҖүз»„д»¶пјҢеҶҚиҜҒжҳҺж–°е®һзҺ°дҝқжҢҒеҺҹжңүиҜӯд№үпјҢе№¶жҸҗеүҚеҶҷеҮәеҒңжӯўдёҺеӣһж»ҡжқЎд»¶гҖӮ

#дҪ•ж—¶дҪҝз”Ё

иҝҷдёҚжҳҜйҰ–дёӘ Agent зҡ„й—ӯеҚ·иҖғиҜ•гҖӮеҸӘжңү TypeScript + Node еҚ• Agent Runtime е·Іиҫҫ L1пјҢе№¶дё”жңү profile/SLO/йғЁзҪІ/йҡ”зҰ»ж•°жҚ®жҢҮеҗ‘дёҖдёӘе…·дҪ“з»„д»¶ж—¶пјҢжүҚеҗҜеҠЁжң¬й—ЁзҰҒгҖӮиӢҘжЁЎеһӢ/з¬¬дёүж–№зҪ‘з»ңеҚ жҚ®дё»иҰҒе»¶иҝҹпјҢжҲ– Node жңӘиҫҫиө„жәҗ/SLO дёҠйҷҗпјҢжӯЈзЎ®з»“и®әеҸҜд»ҘжҳҜвҖңдёҚиҝҒз§»вҖқгҖӮ

#зЎ¬жҖ§еүҚзҪ®

  • L1 Eval гҖҒе®үе…ЁдёҚеҸҳйҮҸгҖҒж•…йҡңжіЁе…ҘгҖҒжҲҗжң¬дёҺе»¶иҝҹеҹәзәҝе·Іеӣәе®ҡгҖӮ
  • е·ІйҖҡиҝҮ profile е®ҡдҪҚеҲ°еҖҷйҖүз»„д»¶пјҢдёҚз”ЁвҖңRust жӣҙеҝ«вҖқд»Јжӣҝж•°жҚ®гҖӮ
  • еҖҷйҖүиҫ№з•ҢеҸҜзӢ¬з«Ӣиҫ“е…Ҙ/иҫ“еҮәгҖҒеҸҜ shadowгҖҒеҸҜеӣһж»ҡпјҢйҰ–йҖүзәҜеҮҪж•°гҖҒparser/indexerгҖҒеҸӘиҜ»е·Ҙе…·жҲ–зӢ¬з«Ӣ sidecarгҖӮ
  • дёҡеҠЎжҺҲжқғгҖҒAgent State е’Ңдә§е“Ғ UX жҡӮд»Қд»Ҙ Node жҺ§еҲ¶йқўдёәжқғеЁҒгҖӮ

#йҳ¶ж®өй—ЁзҰҒ

#R0 В· иҜӯиЁҖдёҺзәҜи®Ўз®—

еҝ…йЎ»иҜҒжҳҺпјҡ

  • иғҪз”Ё ownership/borrow/lifetime и§ЈйҮҠеҖјдёҺ API иҫ№з•ҢгҖӮ
  • з”Ё Option/Result/enum/match/trait е»әжЁЎпјҢж— жңӘи§ЈйҮҠ unwrap/panicгҖӮ
  • tests + Clippy + formatter йҖҡиҝҮпјӣTS/Rust з”ЁеҗҢдёҖ golden fixtures еҜ№жӢҚгҖӮ
  • е…ҲиҝҒз§»ж— еүҜдҪңз”Ё reducer/parserпјҢз»“жһңдёҺй”ҷиҜҜеҲҶзұ»зӯүд»·гҖӮ

иҫҫж ҮеҗҺжүҚиҝӣ R1гҖӮ

#R1 В· ејӮжӯҘдёҺеҸӘиҜ»жү§иЎҢ

еҝ…йЎ»иҜҒжҳҺпјҡ

  • иғҪеҢәеҲҶ Tokio taskгҖҒOS threadгҖҒspawn_blocking дёҺзӢ¬з«ӢиҝӣзЁӢгҖӮ
  • bounded channel/semaphoreгҖҒtimeoutгҖҒcancellation safetyгҖҒgraceful shutdown еқҮжңүж•…йҡңжөӢиҜ•гҖӮ
  • еҸӘиҜ»е·Ҙе…·еңЁе№¶еҸ‘дёҺеҸ–ж¶ҲдёӢж— жі„жјҸд»»еҠЎгҖҒж— ж— з•ҢйҳҹеҲ—гҖҒTrace иҝһз»ӯгҖӮ
  • CPU/йҳ»еЎһе·ҘдҪңдёҚеҚ з”Ё async executorгҖӮ

#R2 В· и·ЁиҜӯиЁҖ Sidecar

еҝ…йЎ»иҜҒжҳҺпјҡ

  • зүҲжң¬еҢ– JSON-RPC/HTTP/gRPC/MCP иҫ№з•ҢдёҺ capability negotiationгҖӮ
  • IDгҖҒйҮ‘йўқгҖҒж—¶еҲ»/ж—¶й•ҝ/deadlineгҖҒmissing/nullгҖҒbytesгҖҒUnicodeгҖҒcanonical hash е…Ёжңү wire-level и§„иҢғе’Ң golden vectorsгҖӮ
  • mTLS/зӯүд»· workload identityгҖҒaudience-bound зҹӯжңҹеҮӯиҜҒгҖҒзӯҫеҗҚ/дёҚйҖҸжҳҺ policy decision referenceгҖҒanti-replay е’Ңиө„жәҗжңҚеҠЎйҮҚж–°жҺҲжқғгҖӮ
  • contract/golden/property tests иҰҶзӣ–зүҲжң¬гҖҒиҫ№з•ҢгҖҒй”ҷиҜҜгҖҒж–ӯжөҒгҖҒеҸ–ж¶ҲгҖҒиҝҹеҲ°е“Қеә”е’Ң unknown effectгҖӮ
  • иҝӣзЁӢиҫ№з•ҢдёҚиў«е®Јз§°дёәиҮӘеҠЁиө„жәҗ/е®үе…Ёйҡ”зҰ»пјӣCPU/еҶ…еӯҳ/ж–Үд»¶/зҪ‘з»ң/еҮӯиҜҒйҷҗеҲ¶еқҮжңүзӢ¬з«Ӣи®ҫзҪ®е’ҢжөӢиҜ•гҖӮ

#R3 В· ShadowгҖҒCanary дёҺз”ҹдә§иҝҒз§»

еҝ…йЎ»иҜҒжҳҺпјҡ

  • Shadow дёҚдә§з”ҹеүҜдҪңз”ЁпјҢжҜ”иҫғ outcome/error/trace/audit/latency/resource/cost иҖҢйқһеҸӘжҜ”ж–Үжң¬гҖӮ
  • еңЁйў„жіЁеҶҢзҡ„ MDE дёҠжҳҫзӨәзңҹе®һж”№е–„пјҢе®үе…ЁгҖҒжӯЈзЎ®жҖ§е’Ңе°ҫе»¶иҝҹж— йҖҖеҢ–гҖӮ
  • Limited canary жңүжөҒйҮҸ/з§ҹжҲ·/е·Ҙе…·иҢғеӣҙгҖҒиҮӘеҠЁеҒңжӯўжқЎд»¶е’Ңиө„жәҗдёҠйҷҗгҖӮ
  • дёҖй”®еӣһж»ҡеҗҺпјҢж—§ Node/Rust з»„еҗҲд»ҚиғҪи§Јз ҒжҢҒд№…дәӢд»¶е’ҢжҒўеӨҚеңЁйҖ” RunгҖӮ
  • е…·дҪ“зҡ„иҝҒз§»жҲҗеҠҹж ҮеҮҶе’ҢвҖңдёҚеҶҚз»§з»ӯиҝҒз§»вҖқеҒңжӯўжқЎд»¶е·ІеҶҷжҳҺгҖӮ

#зЎ¬еӨұиҙҘ

еҮәзҺ°д»»дёҖйЎ№еҲҷеҒңжӯўиҝҒз§»пјҡ

  • ж—  profile/SLO/йғЁзҪІиҜҒжҚ®пјҢзҗҶз”ұеҸӘжҳҜвҖңRust жӣҙеҝ«/жӣҙе®үе…ЁвҖқгҖӮ
  • з”Ё float дј йҮ‘йўқпјҢз”Ё JS number дј дёҚйҖҸжҳҺ IDпјҢжҲ–жңӘе®ҡд№ү missing/null/canonical hashгҖӮ
  • Rust жү§иЎҢйқўдҝЎд»» Node дј жқҘзҡ„ authorized=true иҖҢж— еҸҜйӘҢиҜҒиә«д»Ҫ/еҶізӯ–еј•з”Ё/иө„жәҗжңҚеҠЎйҮҚж–°жҺҲжқғгҖӮ
  • жҠҠеҶ…еӯҳе®үе…ЁжҲ–иҝӣзЁӢиҫ№з•ҢеҪ“дҪңжІҷз®ұгҖӮ
  • зјәе°‘ contract testsгҖҒshadow еүҜдҪңз”ЁзҰҒжӯўгҖҒcanary еҒңжӯўжқЎд»¶жҲ–еҸҜйӘҢиҜҒ rollbackгҖӮ
  • дёәиҝҒз§»ж”№еҶҷ Agent иҜӯд№ү/жҺҲжқғ/иҜ„жөӢпјҢеҚҙж— зӢ¬з«ӢдёҡеҠЎйңҖжұӮе’ҢиҜҒжҚ®гҖӮ

#йҖҡиҝҮеҗҺзҡ„еҺҹеҲҷ

еҸӘиҝҒз§»е·ІиҜҒжҳҺеҸ—зӣҠзҡ„иҫ№з•ҢгҖӮTypeScript + Node еҸҜд»Ҙй•ҝжңҹдҝқз•ҷдёәдә§е“ҒдёҺ Agent жҺ§еҲ¶йқўпјӣвҖңжңҖз»Ҳе…ЁйғЁ RustвҖқдёҚжҳҜй»ҳи®Өзӣ®ж ҮгҖӮ

#жң¬з« е°Ҹз»“

Rust иҝҒз§»йҖҡиҝҮзҡ„ж Үеҝ—дёҚжҳҜд»Јз ҒйҮҸпјҢд№ҹдёҚжҳҜеұҖйғЁ benchmarkпјҢиҖҢжҳҜ outcomeгҖҒTraceгҖҒе®үе…ЁгҖҒе°ҫе»¶иҝҹе’Ңеӣһж»ҡиғҪеҠӣе…ұеҗҢж»Ўи¶ійў„жіЁеҶҢж ҮеҮҶгҖӮд»»дҪ•йҳ¶ж®өиҜҒжҚ®дёҚи¶іпјҢйғҪеә”еӣһеҲ° и·ЁиҜӯиЁҖеҘ‘зәҰ жҲ–зӣҙжҺҘеҒңжӯўиҝҒз§»гҖӮ
